Google Places API (New) from Go and the terminal

goplaces

A modern CLI and typed Go client for finding places, resolving locations, fetching details and photos, autocompleting input, and working with route-aware searches.

Command docs

Each command supports human-readable output by default and stable JSON with --json. Global flags work everywhere.

Places Text Search

 Nearby Search

Setup

Install, key, run.

Enable Places API (New). Enable Routes API for route and directions. Then export one API key.

Install

Use Homebrew for the CLI or go install for a direct Go toolchain install.

shell
brew install steipete/tap/goplaces
go install github.com/steipete/goplaces/cmd/goplaces@latest
export GOOGLE_PLACES_API_KEY="..."
Go package

Same workflows, typed.

The root package is github.com/steipete/goplaces. Requests mirror the CLI concepts and own their field masks.

go
client := goplaces.NewClient(goplaces.Options{
    APIKey:  os.Getenv("GOOGLE_PLACES_API_KEY"),
    Timeout: 8 * time.Second,
})

response, err := client.Search(ctx, goplaces.SearchRequest{
    Query: "italian restaurant",
    Limit: 10,
    LocationBias: &goplaces.LocationBias{Lat: 40.8065, Lng: -73.9719, RadiusM: 3000},
})
