Places Autocomplete

autocomplete

Return place and query suggestions from partial input, with session tokens and optional location bias for billing-friendly flows.

searchnearbyautocompletedetailsphotoresolveroutedirections

Usage

goplaces autocomplete <input> [flags]

shell
goplaces autocomplete "cof" --session-token "goplaces-demo" \
  --limit 5 --language en --region US

Flags

--limitMax suggestions, 1-20. Default: 5.
--session-tokenSession token for billing consistency across autocomplete and details.
--language, --regionBCP-47 language and CLDR region hints.
--lat, --lng, --radius-mOptional location bias. Provide all three together.

Location Bias

goplaces autocomplete "pizza" --lat 40.7411 --lng -73.9897 --radius-m 1500
