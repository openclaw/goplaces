Place Details

details

Fetch rich place data by place ID, including address, coordinates, contact links, opening hours, status, optional reviews, and optional photos.

searchnearbyautocompletedetailsphotoresolveroutedirections

Usage

goplaces details <place_id> [flags]

shell
goplaces details ChIJN1t_tDeuEmsRUsoyG83frY4 --reviews --photos

Flags

--language, --regionBCP-47 language and CLDR region hints.
--reviewsInclude reviews in the response.
--photosInclude photo metadata in the response.

Pair With

Use photo with a returned photo resource name to get a media URL.
