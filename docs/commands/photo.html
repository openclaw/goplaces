Photo Media

photo

Convert a Places photo resource name into a media URL, optionally constrained by maximum width or height.

searchnearbyautocompletedetailsphotoresolveroutedirections

Usage

goplaces photo <photo_name> [flags]

shell
goplaces photo "places/PLACE_ID/photos/PHOTO_ID" --max-width 1200

Flags

--max-widthMaximum returned image width in pixels, 1-4800.
--max-heightMaximum returned image height in pixels, 1-4800.

Source Names

Fetch metadata first with details --photos, then pass one returned places/.../photos/... resource name here. Provide at least one size flag.
